1) Замени userdata.txt на свой файл.
2) Создай бота у @BotFather.
3) В Railway добавь переменную BOT_TOKEN со своим токеном.
4) Загрузи эти файлы в GitHub.
5) В Railway -> New Project -> Deploy from GitHub Repo.
6) Если бот не стартует автоматически, укажи Start Command: python bot.py

Важно: токен не вставляй в bot.py. Храни его только в переменных Railway.
